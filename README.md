## рҹҡҖ GETTING STARTED

### Step 1: Accept the Assignment
1. Click the GitHub Classroom invitation link (provided in Discord)
2. Accept the assignment
3. GitHub will create your personal repository: `week-02-labs-tickets-YOUR-USERNAME`

### Step 2: Clone Your Repository
```bash
# Navigate to your Bootcamp folder
cd Desktop/Bootcamp

# Clone your personal repo (replace YOUR-USERNAME)
git clone https://github.com/3LOGY-Bootcamp-2026/week-02-labs-tickets-YOUR-USERNAME.git

# Navigate into the folder
cd week-02-labs-tickets-YOUR-USERNAME
```

### Step 3: Complete the Assignments
For each lab/ticket:
1. **Read the instructions:** Open `LAB-XX/instructions.md` or `TICKET-XX/requirements.md`
2. **Write your code:** Create your `.js` file inside the `code/` folder
3. **Test your code:** Run it and verify it works
4. **Commit frequently:** Don't wait until Friday!
5. **Check submission requirements:** Review the `submission.md` checklist
