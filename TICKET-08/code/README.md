# Data Structure Coding Challenges

This project solves 15 coding challenges using JavaScript data structures such as Arrays, Objects, and Sets.

## Approach

Each problem is solved using simple JavaScript functions.

### Arrays

Arrays were used for:

* Removing duplicates using Set
* Rotating arrays using slice and concat
* Flattening nested arrays using flat
* Sorting to find the second largest value

### Objects

Objects were used for:

* Counting character frequencies
* Grouping items by properties
* Storing key-value pairs like dictionaries

### Sets

Sets were used to:

* Remove duplicates
* Find union of arrays
* Find intersection of arrays
* Find differences between arrays

### Mixed Challenges

Two small applications were implemented:

1. Phone Book
   Stores contacts and allows searching by name.

2. Shopping Cart
   Stores items with quantities and allows adding/removing products.

## Running the Code

Run the program using Node.js:

```
node data-challenges.js
```

All functions include simple test cases printed to the console.
